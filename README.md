# django-api-server-template

A template for creating a Django API server with REST framework, JWT authentication, and CORS support.

## Features

- Django REST Framework for building APIs
- JWT Authentication using `djangorestframework-simplejwt`
- CORS support with `django-cors-headers`
- Pre-configured settings for quick setup
